# Sales Performance Analysis Dashboard

## Problem Statement
Analyze sales data to identify trends, top-performing products, and regional performance.

## Tools Used
- Excel
- Power BI
- SQL

## Project Outcomes
- Cleaned and transformed raw sales data for analysis.
- Built interactive Power BI dashboards showing monthly trends and KPIs.
- Identified top revenue-generating products and regions to support business decisions.

## KPIs
- Total Sales
- Monthly Sales Growth
- Top Products by Revenue
- Region-wise Performance

## Business Value
This dashboard enables stakeholders to monitor performance, identify growth opportunities, and make data-driven decisions.
